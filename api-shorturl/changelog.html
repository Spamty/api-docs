This are the changes of the Short URL API.
Go to Current version.

v3.1

  • Removed "jsonp" output format

v3

Introducing the original YOURLS' API for 3q3.de.

  • No HTTP GET requests
  • Without the actions "url-stats" (get stats about one short URL), "stats" (get stats about your links) and "db-stats" (get global link and click count)
  • Removed "jsonp" output format

v2.1

We previously used an adapted version of the YOURLS API to create and view shortened URLs which was publicly available. Requests were processed by Spamty API which had access to the YOURLS API.
These versions are now deprecated.

  • xml support
  • Version is returned

v2

  • You have to authenticate with a username and API key.
  • API requests are logged in our database for more security.
  • API is only available with HTTPS.
  • Some smaller improvements.

v1

Introducing this API.
