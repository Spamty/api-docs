Open new ticket

Internal feature requests and issues should be opened in "Development API" or "Development Website".

The Tickets have to be assigned to the user dev@spamty.eu.

Close the ticket

The ticket can be closed if problem was resolved (close as resolved) or if it won't be fixed (just close).

Fix should be described in the ticket for better documentation.
