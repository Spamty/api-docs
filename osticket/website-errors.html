Encrypt

Error

The error code could look like 
encrypt Error: Array ( [emailad] => test@example.com [captcha_code] => ABC [Name] => [submit] => GET LINK )
The Array is $_POST.
There is automatically a new ticket created for the development team containing this error and the status details from API.

Possible Problems

A possible problem could be a error with the API connection in /includes/functions_v4.php.

Fix

  • Check if encryption works generally.
  • Check API logs in database for the error message.

Decrypt

The error code could look like 
decrypt Error: SOME MESSAGE Array ( [lang] => de [version] => v4 [email] => 1 [key] => adbb098612367 )
Firt indicates language of spamty.eu website. Then the version, Email ID in our database and the encryption key. The Array is $_GET.
SOME MESSAGE contains optional additional information.
There is automatically a new ticket created for the development team containing this error.

Possible Problems

If the status message says "Email returned -" the API returned a status of 1 (everything ok) but the key seems to be wrong since the decrypted email address is "-". This happens for example here: http://spamty.eu/mail/v4/20/869656557u8375992x/.

Fix

  • Check if decryption works generally.
  • Check API logs in database for the error message.
