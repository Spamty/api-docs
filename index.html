You can find the documentation for the current api version here. We offer the following methods:

  • Generate a link to an email address
  • View an email address from a spamty link
  • Generate a 3q3.de short URL
  • View long URL from a short URL

You can also use our iframe to show the captcha and your email address directly on your own website.
