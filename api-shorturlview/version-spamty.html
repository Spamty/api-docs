With this API method you can get the long URL from a 3q3.de short URL.

This version is deprecated. Use the latest one.

URL

The API is located here:

https://api.spamty.eu/shorturlview/v2.php

You have to use HTTPS.

Parameters

Send a HTTP request with the following POST parameters:

shortUrl
A 3q3.de short URL (like http://3q3.de/demo).

apiUser
Your class SV API username.

apiKey
Your class SV API Auth Code.

output
The output type (xml or json is supported, default is json).

Example Request

Example request with PHP and cURL

// Init a cURL session
$ch = curl_init();
// Set the URL
curl_setopt($ch, CURLOPT_URL, "https://api.spamty.eu/shorturlview/v2.php");
// No header in the result
curl_setopt($ch, CURLOPT_HEADER, 0);
// Return, do not echo result
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true); 
// This has to be a POST request
curl_setopt($ch, CURLOPT_POST, true);
// ignore the invalid SSL certificate
curl_setopt($ch, CURLOPT_SSL_VERIFYPEER, false); 
// Define the data to POST
curl_setopt($ch, CURLOPT_POSTFIELDS, array(
	// the link
	'shortUrl'	=> 'http://3q3.de/demo',
	// api username & key
	'apiUser'	=> 'sv_931836481',
	'apiKey'	=> '12345678901234541678901234567890ab',
	// output type (xml or json)
	'output'	=> 'json'
));
// Execute and get the response
$response = curl_exec($ch);
// close cURL session
curl_close($ch);

Return Values

The request will return a JSON file with the following values:

status
1 for OK and 0 if any error occurred.

statusDetails
A description if there were any errors

info
The URL where you can get some additional information (this page)

version
The version number.

longUrl
The long URL / The Link to the email address.

Example Response

The response could look like:

JSON

{
   "status" : 1,
   "statusDetails" : "ok",
   "version" : "2.1",
   "info" : "http://dev.spamty.eu/shorturlview",
   "longUrl" : "http://spamty.eu/mail/v4/20/869656557u8375992c/"
}

XML

<spamty>
  <info>https://dev.spamty.eu/shorturlview/</info>
  <version>2.1</version>
  <status>1</status>
  <statusDetails>ok</statusDetails>
  <longUrl>https://spamty.eu/mail/v4/20/869656557u8375992c/</longUrl>
</spamty>
