Feel free to contact me if you have any questions about the Spamty API.

How does authentication work?

Simply send the username (apiUser) and your API key (apiKey) as HTTP POST parameters. You can get them here.

Can I use an API that is marked as "only internal use"?

These APIs are really only for internal use. You cannot use them if you are not part of the Spamty team.

Is it possible to bypass CloudFlare for API requests?

All requests to our API are routed through the CloudFlare system. This makes the API much more faster and protects our servers.
It is possible to bypass CloudFlare if you use api-raw.spamty.eu as domain. But be careful we cannot guarantee that this domain will stay available forever. We may take it down without any announcement. Furthermore the response time is more than twice as long as it is with CloudFlare.
