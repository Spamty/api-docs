Feel free to contact me if you have any questions about the Spamty API.

How does authentication work?

Simply send the username (apiUser) and your API key (apiKey) as HTTP POST parameters. You can get them here.

Can I use an API that is marked as "only internal use"?

These APIs are really only for internal use. You cannot use them if you are not part of the Spamty team.
