The API will return one of these HTTP status codes:

200 OK

The response for a successful request.

400 Bad Request

The status 400 is returned if there was any error with the API. You will get a detailed information from the returned error details.

403 Forbidden

You'll get a 400 Bad Request error code if authorization with API username and key failed. The 403 Forbidden response is used if the API is only for internal use.

404 Not Found

This code is used when the API is not available. If it was available here but is now deprecated you will get a 410 Gone response.

You may have typed in the wrong URL.

410 Gone

This code is used when the API is outdated and doesn't work anymore. You should look at the documentation to find the latest version.

500 Internal Server Error

If there were any technical problems with our server you will get a 500 Internal Server Error response. This was not your fault, but you can contact us so we have a look for a fix.
