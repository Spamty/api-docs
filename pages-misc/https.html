HTTPS

You have to use use HTTPS for your API requests.

The domain api.spamty.eu has no verified SSL-certificate so you may get an error message.

Fix

If you are using PHP with curl you can add this to ignore the invalid certificate:

// ignore the invalid SSL certificate
curl_setopt($ch, CURLOPT_SSL_VERIFYPEER, false);
