API Keys

We have multiple classes of API keys (you can find them below).

You'll have to request an API key via email. Please provide following information:

  • Your name/company
  • Your website/application website
  • Your email address
  • What you want to do with the API
  • If this is a commercial project
  • How many API requests you expect to send per week/day/...

We will check your request and tell you if you are eligible to use this API.

Class E

This API key is used to generate a link to an email address with spamty.

See: API v3 Encryption

Class D

This key has to be used when you want to see an email address from a spamty link.

See: API v3 Decryption

Class SC

Class SC is to generate a short URL with 3q3.de.

See: Create a short URL

Class SV

Class SV is to view a spamty link from a 3q3.de short url.

See: View a short URL
