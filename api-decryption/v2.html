This is the old version 2.2. Go to the current one and see what changed.

URL

This API is located at:

https://api.spamty.eu/decryption/v2.php

Authentication

It is only for internal use. You need an additional authentication with HTTP Basic Auth.

Parameters

Send a HTTP POST request to the URL with the following POST parameters:

n
The apiN parameter of the encryption request (a number).

m
The apiM parameter of the encryption request (string with a-z A-z 0-9).

k
The apiK parameter of the encryption request (string with a-z A-z 0-9).

apiUser
Your class D Username.

apiKey
Your class D API Auth Code.

output
The output type ("xml" or "json" is supported)

Return Values

The request will return a JSON file with the following values:

status
1 for OK and 0 if an error occurred.

statusDetails
A description if there were any errors. Otherwise the value is “ok”.

info
The URL where you can get some additional information (this page)

version

email
The email address

mailCheck
A crc32b hash of the email address
