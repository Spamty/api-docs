This is the old documentation for version 1. Go to the latest version and see what changed.

This version is deprecated and doesn't work anymore!

URL

The API is located at:

https://api.spamty.eu/encryption/v1.php

Parameters

You have to pass the following parameters with a HTTP POST request the API:

theDomain
The domain (example.com if email address is test@example.com).

theEmailA
The mail name (test if email address is test@example.com).

apiKey
Your class E API Key

output
The output type (only json is supported)

Return Values

The request will return a JSON file with the following parameters:

status
Value 1 for OK and 0 if an error occurred.

apiN

apiM

apiK

urlPublic
The public decryption url which can be shared.

mailCheck
A md5 hash of the email address.
