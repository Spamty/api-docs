With this API method you can generate a Spamty link to an email address.

Current version: 4.1. See what changed.

URL

The API is located at:

https://api.spamty.eu/encryption/v4.php

You have to use HTTPS, but our SSL certificate is not validated so you might get an error message.

Parameters

You have to pass the following parameters with a HTTP POST request the API:

email
The complete email address (test@example.com).

apiUser
Your class E API Username

apiKey
Your class E API Key

output
The output type (json or xml is supported, default is json)

Return Values

The request will return a JSON file with the following parameters:

status
Value 1 for OK and 0 if an error occurred.

statusDetails
A description if there were any errors. Otherwise the value is ok.

info
The URL where you can get some additional information about the API (this page).

version
The version number.

apiM
The m parameter for the decryption request. This is a number.

apiK
The k parameter for the decryption request. A string with no special chars.

urlPublic
The public decryption url which can be shared (like https://spamty.eu/mail/v4/20/869656557u8375992c/ ).

mailCheck
A crc32b hash of the email address
