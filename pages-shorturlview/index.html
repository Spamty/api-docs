View a short URL

With this API method you can get the long URL from a 3q3.de short URL.

The current version is 2.1 (see what changed).

URL

The API is located here:

https://api.spamty.eu/shorturlview/v2.php

You have to use HTTPS.

Parameters

Send a HTTP request with the following POST parameters:

shortUrl
A 3q3.de short URL (like http://3q3.de/demo).

apiUser
Your class SV API username.

apiKey
Your class SV API Auth Code.

output
The output type (xml or json is supported, default is json).

Return Values

The request will return a JSON file with the following values:

status
1 for OK and 0 if any error occurred.

statusDetails
A description if there were any errors

info
The URL where you can get some additional information (this page)

version
The version number.

longUrl
The long URL / The Link to the email address.

The response could look like:

{
   "status" : 1,
   "statusDetails" : "ok",
   "version" : "2.1",
   "info" : "http:\/\/dev.spamty.eu\/shorturlview",
   "longUrl" : "http:\/\/spamty.eu\/mail\/v4\/20\/869656557u8375992c\/"
}
